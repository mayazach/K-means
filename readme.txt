Όνομα εκτελέσιμου προγράμματος: kmeans

Το πρόγραμμα μεταγλωτίζεται με την εντολή make, και καλείται από την γραμμή εντολής με τις εξής σημαίες σε οποιαδήποτε σειρά:
-i <input file>
-c <config file>
-o <output file>
-d <DFT/DTW>
-complete (προεραιτική)

Οι άλλες τιμές που έχουν χρησημοποιηθεί στο πρόγραμμα είναι:
μέγεθος hash table = n/4

Επειδή το πρόγραμμα είναι πολύ αργό όταν χρησιμοποιείται το πλήρες dataset, χρησιμοποιήσαμε για τις δοκιμες ένα μικρότερο dataset με 100 καμπύλες του αρχικού.
Καθώς εκτελείται, τυπώνει στην έξοδο το συνδυασμό που ξεκινάει να δοκιμάζει, καθώς και τον αριθμό των κέντρων που άλλαξαν σε κάθε update.
Το stopping criterion στον αλγόριθμο PAM είναι όταν κανένα κέντρο δεν έχει αλλάξει, κάτι που συμβαίνει πάντα αργά η γρήγορα. Στον αλγόριθμο υπολογισμού Discrete Mean Frechet auto δε συμβαίνει πάντα οποτε υπάρχει το επιπλέον κριτήριο να σταματάει μετά από έναν αριθμό επαναλήψεων, αν τα κέντρα δεν έχουν σταθεροποιηθεί.
Από τις δοκιμες μέχρι στιγμής, η ανάθεση του Lloyd δίνει πολύ καλύτερα αποτελέσματα από την LSH όταν χρησιμοποιείται η συνάρτηση απόστασης DFT, ενώ η LSH είναι καλύτερη για τη συνάρτηση απόστασης DTW.

-curve.h: αντικείμενο της καμπύλης, με τα σημεία και το id της, καθώς και αν ανήκει στο dataset, αν ανήκει σε cluster  το id του αν ναι (για τη συνάρτηση lsh)
